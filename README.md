# 📘 Student Grade Tracker System

## 📌 Project Overview

The Student Grade Tracker is a console-based Java application developed to efficiently manage and analyze student academic performance. This project is designed to help users store student names and marks, perform calculations on the entered data, and generate a complete performance summary report.

The application demonstrates the practical implementation of Java programming fundamentals, including Object-Oriented Programming (OOP), ArrayLists, loops, conditional statements, user input handling, and data processing techniques.

This project is ideal for beginners who want to strengthen their understanding of Java concepts through a real-world mini project.

---

# 🎯 Project Objective

The primary objective of this project is to create a simple and effective grade management system that can:

- Collect student details from users
- Store student records dynamically
- Calculate academic statistics automatically
- Display performance reports clearly
- Demonstrate Java programming concepts practically

The system aims to reduce manual calculation efforts and provide a structured way to manage student marks efficiently.

---

# 🌟 Main Features

## ✅ Student Data Management
The system allows users to enter and manage multiple student records dynamically.

## ✅ Dynamic Storage Using ArrayList
Student details are stored using Java ArrayLists, which provide flexible and dynamic data handling.

## ✅ Automatic Calculations
The application automatically calculates:
- Average marks
- Highest marks
- Lowest marks

## ✅ Summary Report Generation
A well-formatted summary report displays:
- Student names
- Individual marks
- Overall class statistics

## ✅ Beginner-Friendly Console Interface
The project uses a simple console-based interface, making it easy to understand and execute.

## ✅ Object-Oriented Programming Implementation
The project uses:
- Classes
- Objects
- Constructors
- Encapsulation concepts

# 🧠 Concepts Implemented

## 🔹 Classes and Objects
A Student class is created to represent student data such as:
- Student Name
- Student Marks

Objects are created for each student and stored in an ArrayList.

---

## 🔹 ArrayList Collection
The ArrayList collection framework is used to:
- Dynamically store records
- Add multiple students
- Iterate through data efficiently

---

## 🔹 Loops
Loops are used for:
- Taking multiple student inputs
- Displaying reports
- Calculating totals and statistics

---

## 🔹 Conditional Statements
Conditional logic is implemented to:
- Find highest marks
- Find lowest marks
- Compare student performance

---

## 🔹 User Input Handling
The Scanner class is used to collect:
- Number of students
- Student names
- Student marks

---

## 🔹 Data Processing
The application processes student data to calculate:
- Total marks
- Average marks
- Performance statistics
